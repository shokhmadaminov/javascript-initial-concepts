JAVASCRIPT Asoslari

    JavaScript "String" Metodlari

  1. toUppercase() - Harflarni katta qilib beradi
  2. toLowercase() - Harflarni kichik qilib beradi
  3. indexOf() - Stringni birinchisini tartib raqamini chiqarib beradi
  4. lastIndexOf() - Stringni oxirgi tartib raqamini chiqarib beradi
  5. slice() - Tartib raqami bo'yicha berilgan sondan keyingi harflarni oladi
  6. substr() - Tartib raqami bo'yicha berilgan sondan keyin nechta harfni olishniyam aytishimiz mumkin
  7. replace() - Stringdagi biron bir harf o'rniga boshqasini qo'yishimiz mumkin
  8. charAt() - Har doim string ichidagi qiymatni birinchi harfini olib beradi
  9. trim() - Stringdagi bo'sh joylarni yo'qotib beradi
  10. split() - Stringni harflarini bo'lishda va x.k.z larda kerak bo'ladi
  11. includes() - Includes bu tekshiruvchi, tarkibda shu harf yoki biron nima bormi yoki yo'qmi bilib beradi
  12. startWith() - Stringni shu harflardan boshlangan yoki yo'qligini tekshirishimiz mumkin!
  13. substring() - String ichidagi biron bir joylarini kesib ajratib olishimiz mumkin!
  14. eval() - Eval metodi sizga kankulyator vazifasini bajarib berishi mumkin. Ya'ni, stringda sonlar ustda matematik amallar bajarilganda evalni ishlatishimiz mumkin.

    JavaScript Propertylari

  1. length - Berilgan so'z yoki obyektni sonini yoki uzunligini aniqlashda ishlatamiz
  2. typeOf - Berilgan ma'lumotni qanaqa turga mansubligini aniqlaydi

    JavaScript matematik amallari

  1. ( + ) - Qo'shish
  2. ( - ) - Ayrish
  3. ( * ) - Ko'paytirish
  4. ( / ) - Bo'lish
  5. ( ** ) - Darajaga ko'tarish
  6. ( % ) - Qoldiqli bo'lish
  7. ( ++ ) - bir dona qo'shib qo'yish
  8. ( -- ) - bir dona ayrib qo'yish

    JavaScript "Array" Metodlar

  1. indexOf() - Array ichidagi ma'lumotlar tartib raqamini chiqarib beradi
  2. concat() - Arraylarni bir-biriga qo'shib beradi
  3. push() - Arrayni oxiriga biron bir element qo'shib beradi
  4. pop() - Arrayni oxiridan biron bir elementni olib tashlaydi
  5. unshift() - Arrayni boshiga biror bir elementni qo'shib beradi
  6. shift() - Arrayni boshidan biror bir elementni olib tashlaydi
  7. reverse() - Arrayni teskarisiga o'girib beradi, misol [a,b], [b,a]
  8. filter() - Array ichidagi turli xil turdagi ma'lumotlarni filterlashda ishlatiladi
  9. find() - Birinchi bo'lib shartni bajargan array elementini qaytaradi. Array ichidagi biror elementni qidirishda ishlatiladi
  10. findIndex() - Birinchi bo'lib shartni bajargan array elementini joylashgan o'rnini qaytaradi. Array ichidagi biror elementni joylashgan joyini topishda ishlatiladi
  11. map() - Har bir massiv elementi uchun berilgan funktsiyani chaqirib, yangi array yaratadi
  12. sort() - Elementlarni alfavit tartibida, satrlar sifatida va o'sish tartibida tartiblaydi
  13. splice() - Massivdagi biror bir element yoki elementlar o'rniga boshqa bir elementni joylashtirishimiz mukin yoki shunchaki massivdagi biror bir elementdan keyinga biror bir elementni qo'shib qo'yishimiz mumkin

    Object Methodlar

  1. Object.assign() - Obyektdagi va arraydagi ma'lumotlarni nusxalab beradi.
  2. Object.entries() - Obyektdagi ma'limotlani array ichidagi arraylarga bo'lib beradi.
  3. Object.keys() - Obyektdagi "key" larni alohida arrayga kiritib beradi.
  4. Object.values() - Obyektdagi "value" larni alohida arrayga kiritib beradi.
